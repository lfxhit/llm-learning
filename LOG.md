## 2026年4月1日 Day2
- 今天实验室的学长交给我一个工作：将一个酒店语音模型在云端部署。这是我第一次完成相关工作，主要是按照学长给的部署文件一步步操作。
  - 但是遇到了一个小问题：能通过公网 IP 访问服务器，前端页面能打开，但语音交互功能无法使用（录音、对话无响应）
  - 其原因是前端页面是 HTTPS（手机浏览器强制），LiveKit Server 提供的是 WS://（未加密 WebSocket），浏览器安全策略阻止了 HTTPS 页面连接 WS 服务。
  - 目前有两个方案：1.给 LiveKit 配置 WSS，2.用 ngrok 快速获得 WSS。在部署之前我们提前配好了2，但是不起作用了。
